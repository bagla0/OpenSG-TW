================================================================================
 RM OpenSG-TW  --  CROSS-SECTION HOMOGENIZATION + DEHOMOGENIZATION  API REFERENCE
================================================================================
 Sphinx-style function reference for the Reissner-Mindlin (RM) thin-walled
 cross-section pipeline: the C0 drilling-constrained 6-DOF ring element, the
 boundary-ring homogenizer, and the two-step dehomogenization (beam force ->
 pointwise 3-D stress).  Every function is listed with its file, signature,
 purpose, arguments, returns, and notes.  (Generated 2026-07-12.)

 Companion docs:  dehom_procedure.txt  (the theory + step-by-step algorithm)
                  rm_8x8_homogenization.txt  (the plate 8x8 line-by-line)


================================================================================
 1.  WHAT THE PIPELINE DOES  (the 30-second picture)
================================================================================
 A wind-blade cross-section is a 1-D closed contour of laminated shell walls.
 MSG (Mechanics of Structure Genome) condenses that 2-D wall problem into a
 beam 6x6 (HOMOGENIZATION), and inverts it to recover pointwise 3-D stress from
 a beam force (DEHOMOGENIZATION).  The shell model is Reissner-Mindlin: a C0
 Lagrange 4-node strip element with 6 DOF/node (3 displacement + 3 rotation,
 including an INDEPENDENT drilling omega_3 fixed by a Lagrange multiplier), and
 MITC-tied gamma_23 transverse shear.

     YAML contour + laminates
        |  load_ring_ref                         (oml_ring.py)
        v
     ring arrays  rx, cells, rsub, re3, D_by, G_by, k22, ax, cross
        |  ring_indep(return_fields=True)        (run_ring_indep.py)
        |     -> assemble_segment_indep          (segment_indep.py)
        |          -> quad_ops_indep              (8-strain element operators)
        |          -> _mitc_shear_indep           (tied gamma_23)
        |     -> assemble_constraint (drilling Lagrange), build_C_Psi (rigid kernel)
        |     -> KKT solve V0; prepare_v1_rhs/finalize_v1_and_compute_deff (msg_solver.py)
        v
     C6_RM (Timoshenko 6x6),  V0,V1 (warping, 6m x 4)
        |  build_rm_bundle                        (dehom_rm.py)   <-- HOMOGENIZE ONCE
        v
     bundle {Timo=C6_RM, V0, V1, strip, corners, red_cells, k22, re3,
             layup_per_elem, layup_db, material_db, frac}
        |  stress_at_points(bundle, points, beam_force_vabs=FF)   (dehom_rm.py)
        |   STEP 1  st=C6_RM^-1 FF; _macro_recovery; s6/s2 = RM element operators
        |   STEP 2  plate_stress_at_depth  (msg_materials.py)  -> in-plane + s33 3-D stress
        |   STEP 3  transverse_shear_stiffness recover (msg_transverse_shear.py)  [diagnostic]
        v
     stress(P,6) = [S11,S22,S33,S23,S13,S12] at every query point

 quad_ops_indep   [public]
................................................................................
  quad_ops_indep(X, e3m, xi, eta, k22, cross, ax, kg=0.0)

  Scalar (single-element) evaluation of the 8-strain 6-DOF C0 drilling-
  independent element operators plus the finite drilling residual DR at one
  quadrature point (xi,eta). This is the closed-form B-operator core for one
  MITC4 quad with omega_3 kept as a genuine 6th nodal DOF.
  Args:
    X              (4,3) float array; the 4 corner node coordinates of the
                   quad element.
    e3m            (3,) float array; the reference material/director e3 used
                   to orient (sign-fix) the surface normal n.
    xi             float; first parametric coordinate in [-1,1].
    eta            float; second parametric coordinate in [-1,1].
    k22            float; edge/curvature parameter passed through (NOT
                   actually used inside these operator rows -- retained for
                   signature symmetry).
    cross          length-2 list/sequence of the two cross-section (in-plane)
                   axis indices (the two components != ax), e.g. [j for j in
                   range(3) if j!=ax].
    ax             int; the beam/span axis index (0,1,2).
    kg             float, default 0.0; geometric/curvature parameter passed
                   through, unused in operator rows.
  Returns:
    10-tuple (BDe, BDh, BDl, BGe, BGh, BGl, DRe, DRh, DRl, dA): BDe (6,4)
    deformation-strain rows on the classical strain params eb; BDh (6,24) on
    the fluctuation w; BDl (6,24) on the derivative/chain-rule term w'; BGe
    (2,4) transverse-shear rows on eb; BGh (2,24) shear on w; BGl (2,24) shear
    chain-rule; DRe (4,), DRh (24,), DRl (24,) the drilling-residual DR rows
    on eb/w/w'; dA float area element. Row order of the 6 deformation strains
    is [k11-membrane e11, e22, 2e12, k11, k22, k12]; the 2 shear rows are
    [2g13, 2g23]. Each node's 6 DOF are [w1,w2,w3,om1,om2,om3].
  Notes:
    Implements the segment-header math: membrane rows unchanged (no om3);
    curvature Lambda_a terms enter DIRECTLY through om3 (dof 5), Lambda_a =
    om3|a + x_{1;a} om3'; transverse shear is pre-elimination with NO 1/C33
    (2g13 += x_{3;2} om3 via +C23, 2g23 += -x_{3;1} om3 via -C13), 'swept' =
    x2*y3 - x3*y2; DR = C33 om3 + C3b om_b - S/2, DRe[1] = -0.5*(x11*Rn2 -
    x12*Rn1) is the twist column, with Rn1=x2*x31-x3*x21, Rn2=x2*x32-x3*x22.
    Calls _surf_frame (from segment_element_general) to build N, D1, D2, dA
    and the direction-cosine dict c. NDOF6=6.

 quad_ops_indep_batch   [public]
................................................................................
  quad_ops_indep_batch(Xe, e3e, xi, eta, cross, ax)

  Batched (vectorized over all elements) evaluation of the same 6-DOF operator
  rows and drilling residual as quad_ops_indep; this is the performance path
  used by the assembly (the scalar version is kept only for diagnostics). k22
  and kg do not enter these operators so are omitted.
  Args:
    Xe             (ne,4,3) float array; all quad corner coordinates.
    e3e            (ne,3) float array; per-element reference directors.
    xi             float; parametric coordinate.
    eta            float; parametric coordinate.
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
  Returns:
    10-tuple (BDe, BDh, BDl, BGe, BGh, BGl, DRe, DRh, DRl, dA): BDe (ne,6,4);
    BDh (ne,6,24); BDl (ne,6,24); BGe (ne,2,4); BGh (ne,2,24); BGl (ne,2,24);
    DRe (ne,4); DRh (ne,24); DRl (ne,24); dA (ne,). Same row semantics and DOF
    ordering as quad_ops_indep, prepended with the element axis.
  Notes:
    Numerically identical closed form to quad_ops_indep but assembled with
    einsum/broadcasting; DOF blocks are built as (ne,row,node,6) tensors then
    reshaped to (ne,row,24). Displacement (w) blocks occupy dof slots 0:3,
    rotation (om) blocks 3:6. DRe[:,1] = -0.5*(x11*Rn2 - x12*Rn1). Calls
    _surf_frame_batch. Used inside assemble_segment_indep, _tie_rows_batch,
    and assemble_constraint's batched path.

 assemble_segment_indep   [public]
................................................................................
  assemble_segment_indep(nodes, quads, subdom, e3s, D_by, G_by, k22_e, cross, ax, kg_e=None, pen=None, pen_beta=0.1, dof_map=None, shear="full", sparse=False)

  Assemble the full 6-DOF drilling-independent element stiffness system for a
  segment/ring, including the finite drilling-residual penalty pen*DR^2,
  producing the six MSG block matrices used by the Timoshenko homogenization
  solve.
  Args:
    nodes          (Nn,3) float array of node coordinates.
    quads          (ne,4) int array of quad connectivity (node indices into
                   nodes).
    subdom         (ne,) int array; per-element subdomain/section id used to
                   look up D_by/G_by.
    e3s            (ne,3) float array; per-element reference director e3.
    D_by           dict {section_id -> (6,6)} ABD deformation stiffness per
                   subdomain.
    G_by           dict {section_id -> (2,2)} transverse-shear stiffness per
                   subdomain.
    k22_e          (ne,) array; per-element edge/curvature parameter
                   (accepted/passed but not used by the operators).
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
    kg_e           optional (ne,) array or None (default); geometric
                   parameter, unused in operator rows.
    pen            optional float or None (default); drilling penalty weight.
                   If None, set to pen_beta*_d_scale(D_by).
    pen_beta       float, default 0.1; multiplier on the D-scaled penalty when
                   pen is None.
    dof_map        optional (Nn,) int array or None (default arange(Nn)); maps
                   node index -> dof-node index (used for the ring where the
                   top row wraps onto the bottom row, producing repeated
                   dofs).
    shear          str, default 'full'; 'full' integrates both shear rows
                   untied, else a tying scheme name
                   ('mitc4_g23','mitc4_both','mitc4_wonly') passed to
                   _shear_batch.
    sparse         bool, default False; if True return the three (ndof x ndof)
                   hh/hl/ll blocks as scipy CSR sparse matrices (Dhe/Dle stay
                   dense).
  Returns:
    6-tuple (Dhh, Dhe, Dee, Dhl, Dll, Dle): Dhh (ndof,ndof) w-w block; Dhe
    (ndof,4) w-eb block; Dee (4,4) eb-eb block; Dhl (ndof,ndof) w-w' block;
    Dll (ndof,ndof) w'-w' block; Dle (ndof,4) w'-eb block. ndof =
    6*(max(dof_map)+1). Dhh/Dhl/Dll are CSR sparse when sparse=True, else
    dense.
  Notes:
    Integrates over the 4 Gauss points at +/-1/sqrt(3). Element energy = BDh^T
    De BDh + BGt^T Gm BGt + pen*DRh DRh (and cross terms with BDe/BGe/DRe and
    BDl/BGl/DRl). De/Gm are gathered per element via a sorted-key lookup table
    (keys=sorted unique subdom, pos map, sdi index). Global dof vector g
    (ne,24) = 6*dof_map[quads] + arange(6). Dense assembly uses np.add.at
    (handles repeated dofs from a wrapped dof_map); sparse path builds COO
    triplets summed by tocsr(). Dhe/Dle always accumulated dense with
    np.add.at. tie rows precomputed via _tie_rows_batch when shear!='full'.
    Calls quad_ops_indep_batch, _shear_batch, _d_scale.

 assemble_constraint   [public]
................................................................................
  assemble_constraint(nodes, quads, subdom, e3s, k22_e, cross, ax, kg_e=None, lam_space="elem", dof_map=None, sparse=False)

  Assemble the weak drilling-constraint (Lagrange-multiplier) operators
  enforcing the drilling residual DR=0 in a chosen multiplier space, for the
  constrained 6-DOF ring/segment solve.
  Args:
    nodes          (Nn,3) float array of node coordinates.
    quads          (ne,4) int array of quad connectivity.
    subdom         (ne,) int array; per-element subdomain id (accepted for
                   signature symmetry; not used to gather stiffness here).
    e3s            (ne,3) float array; per-element reference director.
    k22_e          (ne,) array; per-element curvature param (only used in the
                   scalar/non-elem fallback path, passed to quad_ops_indep).
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
    kg_e           optional (ne,) array or None (default); geometric param,
                   used only in scalar fallback.
    lam_space      str, default 'elem'; multiplier space: 'elem' (one
                   piecewise-constant multiplier per element, inf-sup stable),
                   'node' (one multiplier per node, equal-order, over-
                   constrains -- ablation), 'elem_nofold' (element-constant
                   but EXCLUDING elements adjacent to a fold line where
                   incident-element normals disagree >30deg).
    dof_map        optional (Nn,) int array or None (default arange(Nn)); node
                   -> dof-node index (wrapped strip in the ring), producing
                   repeated dofs handled via np.add.at.
    sparse         bool, default False; if True return G and Gl as scipy CSR
                   sparse (elem path only).
  Returns:
    3-tuple (G, Gl, Ge): G (P, 6*Nd) constraint operator on the fluctuation w;
    Gl (P, 6*Nd) on the derivative w'; Ge (P, 4) on the classical strains eb.
    P = number of multiplier rows (= number of kept elements for
    elem/elem_nofold, = Nd for node); Nd = max(dof_map)+1; 6*Nd = M. G/Gl are
    CSR sparse when sparse=True (elem path).
  Notes:
    For 'elem'/'elem_nofold' each row is the element-integrated DR (sum over 4
    GPs of DRh*dA, DRl*dA, DRe*dA) via a batched quad_ops_indep_batch loop;
    elem_nofold zeroes/skips elements touching a fold node (fold detected by
    |n_i . n_j|<cos(30deg) among elements sharing a node, per-element normal
    from the two diagonals), remapping kept rows with row_of. For 'node' the
    fallback loop weights each element DR by shape function N[a] into that
    node's multiplier row using the scalar quad_ops_indep. Uses np.add.at for
    repeated-dof safety. Calls quad_ops_indep_batch (elem path),
    quad_ops_indep and _bilinear (node path).

 build_C_Psi_segment6   [public]
................................................................................
  build_C_Psi_segment6(nodes, quads, cross)

  Build the 6-DOF rigid-body constraint matrix C and null-space/rigid-mode
  matrix Psi for the segment, embedding the 4 classical rigid modes (3
  translations + 1 rotation) with the drilling om3 column left at zero
  (drilling is not a rigid mode).
  Args:
    nodes          (Nn,3) float array of node coordinates.
    quads          (ne,4) int array of quad connectivity.
    cross          length-2 list; the two in-plane axis indices (used to pick
                   y2=nodes[:,cross[0]], y3=nodes[:,cross[1]]).
  Returns:
    2-tuple (C, Psi): C (4, ndof) mass/area-weighted constraint rows; Psi
    (ndof, 4) rigid-mode basis. ndof = 6*Nn.
  Notes:
    C rows cc=0..3 place the area-weighted nodal weight node_w on dof 6n+cc;
    node_w accumulated by integrating the shape function * dA over the 4 Gauss
    points via np.add.at. Psi columns 0..2 are the three translations (unit on
    dofs w1,w2,w3 = 6n+0..2); column 3 is the cross-section rotation:
    Psi[6n+1,3]=-y3, Psi[6n+2,3]=y2, Psi[6n+3,3]=-1 (rotation couples w2,w3
    and om1). om3 (dof 5) column stays 0. Calls _bilinear.

 _mitc_shear_indep   [private/internal]
................................................................................
  _mitc_shear_indep(X, e3m, xi, eta, k22, cross, ax, kg=0.0, scheme="mitc4_g23")

  Scalar assembly of the tied (assumed-strain) transverse-shear rows BGt (2 x
  24) at (xi,eta) using Dvorkin-Bathe MITC4 tying, selecting which of the two
  shear rows are tied via the scheme.
  Args:
    X              (4,3) float array; quad corner coordinates.
    e3m            (3,) float array; reference director for normal sign-fix.
    xi             float; parametric coordinate.
    eta            float; parametric coordinate.
    k22            float; curvature parameter, passed to quad_ops_indep
                   (unused in rows).
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
    kg             float, default 0.0; passed through.
    scheme         str, default 'mitc4_g23'; one of 'mitc4_g23' (tie only
                   gamma_23 row), 'mitc4_both' (tie both rows -- naive full
                   tying, aliases drilling shear, ablation), 'mitc4_wonly'
                   (tie both rows at DB points but keep every rotation column
                   om1/om2/om3 at its Gauss-point full-integration value --
                   field-consistent partial tying).
  Returns:
    BGt: (2,24) float array; the tied transverse-shear B-rows [tied g13; tied
    g23] at (xi,eta).
  Notes:
    gamma_23 (row index 1) is always tied at points (0,-1),(0,+1) with linear-
    in-eta interpolation 0.5(1-eta)r0+0.5(1+eta)r1; gamma_13 is tied at
    (-1,0),(+1,0) linear-in-xi only for schemes mitc4_both/mitc4_wonly,
    otherwise taken at the Gauss point. For mitc4_wonly, after vstack the
    rotation columns (dofs 3:6 of each node) are overwritten with the fully-
    integrated Gauss-point BGh values. Uses module tying-point constants
    _TIE_G13 and _TIE_G23. Calls quad_ops_indep (indexes result [4] = BGh).
    ops[4] is the Gauss-point BGh.

 _surf_frame_batch   [private/internal]
................................................................................
  _surf_frame_batch(Xe, e3e, xi, eta, cross, ax)

  Vectorized (batched over elements) version of _surf_frame: builds the local
  orthonormal surface frame, shape-function surface derivatives, area element,
  and direction-cosine dict for ALL elements at one quadrature point.
  Args:
    Xe             (ne,4,3) float array; corner coordinates of all ne quads.
    e3e            (ne,3) float array; per-element reference director for
                   normal sign-fix.
    xi             float; parametric coordinate.
    eta            float; parametric coordinate.
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
  Returns:
    5-tuple (N, D1, D2, dA, c): N (4,) bilinear shape functions (same for all
    elements at this GP); D1 (ne,4), D2 (ne,4) the surface derivative
    operators (= inv(G^T) applied to the parametric shape derivatives); dA
    (ne,) area element per element; c dict of (ne,) direction-cosine component
    arrays with keys x11,x21,x31 (a1 components), x12,x22,x32 (a2 components),
    y1,y2,y3 (normal n components), and x2,x3 (in-plane position components).
  Notes:
    Builds a2 from normalized Jxi, a1 as Jeta orthogonalized against a2 then
    normalized, n = a1 x a2; flips n and a1 where n.e3e < 0. Metric
    G=[[G11,G12],[G21,G22]] with G_ab from a_i.J; [D1;D2] = inv(G^T)[dNx;dNe]
    using det = G11*G22-G12*G21. dA = |Jxi x Jeta|. Calls _bilinear (from
    segment_element) which returns N, dNx, dNe. The dict labels components by
    ax (axial) and cross[0]/cross[1].

 _tie_rows_batch   [private/internal]
................................................................................
  _tie_rows_batch(Xe, e3e, cross, ax)

  Precompute the batched transverse-shear B-rows at the four Dvorkin-Bathe
  tying points once per element set (they are independent of the running
  quadrature point, so this avoids re-evaluating them at every Gauss point).
  Args:
    Xe             (ne,4,3) float array; quad corner coordinates.
    e3e            (ne,3) float array; per-element reference directors.
    cross          length-2 list; in-plane axis indices.
    ax             int; span axis index.
  Returns:
    dict with keys 'g13m','g13p','g23m','g23p', each a (ne,1,24) array:
    g13m/g13p are the gamma_13 row (BGh row 0) at tying points (-1,0)/(+1,0);
    g23m/g23p are the gamma_23 row (BGh row 1) at (0,-1)/(0,+1).
  Notes:
    Calls quad_ops_indep_batch at the four tying points and slices BGh (result
    index [4]) rows 0:1 or 1:2. Consumed by _shear_batch.

 _shear_batch   [private/internal]
................................................................................
  _shear_batch(xi, eta, scheme, BGh_gauss, tie)

  Batched construction of the tied transverse-shear rows at running point
  (xi,eta) from the precomputed tying-point rows; the vectorized mirror of
  _mitc_shear_indep.
  Args:
    xi             float; parametric coordinate.
    eta            float; parametric coordinate.
    scheme         str; tying scheme, one of
                   'mitc4_g23','mitc4_both','mitc4_wonly' (any other value
                   leaves g13 at Gauss).
    BGh_gauss      (ne,2,24) float array; the fully-integrated Gauss-point
                   shear rows at (xi,eta) (BGh from quad_ops_indep_batch).
    tie            dict from _tie_rows_batch with keys g13m,g13p,g23m,g23p,
                   each (ne,1,24).
  Returns:
    BGt: (ne,2,24) float array; tied shear rows [g13; g23] per element.
  Notes:
    g23 = 0.5(1-eta)*g23m + 0.5(1+eta)*g23p always; g13 interpolated
    0.5(1-xi)*g13m+0.5(1+xi)*g13p only for mitc4_both/mitc4_wonly, else taken
    from BGh_gauss[:,0:1,:]. For mitc4_wonly the rotation columns (per node
    dofs 3:6) are overwritten with BGh_gauss values (copies BGt first).
    NDOF6=6 used to build the rotation mask.

 _d_scale   [private/internal]
................................................................................
  _d_scale(D_by)

  Return the characteristic ABD stiffness magnitude = the max over layups of
  max|diag(D)|, used to set a dimensionally-commensurate drilling penalty
  weight.
  Args:
    D_by           dict-of-(6,6) or list-of-(6,6) ABD/stiffness matrices keyed
                   by (or indexed by) section/layup id.
  Returns:
    float; max over all keys of max(|diagonal(D_by[k])|).
  Notes:
    Handles both dict (iterates .keys()) and sequence (iterates range(len))
    inputs. Used by assemble_segment_indep to set pen = pen_beta *
    _d_scale(D_by) when pen is None.

 ring_indep   [public]
................................................................................
  ring_indep(rx, rcells, rsub, re3, D_by, G_by, k22_edge, ax, cross, h=None, shear="mitc4_g23", lam_space="elem", return_fields=False)

  Solve the constrained 6-DOF (independent-drilling, element-Lagrange)
  boundary-ring cross-section SG and return the ring Timoshenko 6x6 stiffness
  C6; optionally also the zeroth/first-order warping fields for the Dirichlet
  segment transfer.
  Args:
    rx             (m,3) float array; ring (contour) node coordinates.
    rcells         (nc,2) int array; ring edge connectivity (each edge = two
                   contour node indices).
    rsub           (nc,) int array; per-edge subdomain/section id.
    re3            (nc,3) float array; per-edge reference director e3 (passed
                   as e3q to the element).
    D_by           dict {section_id -> (6,6)} ABD stiffness.
    G_by           dict {section_id -> (2,2)} transverse-shear stiffness.
    k22_edge       (nc,) array; per-edge k22 curvature parameter.
    ax             int; span axis index.
    cross          length-2 list; in-plane axis indices.

                   None, set to the mean edge length of the ring.
    shear          str, default 'mitc4_g23'; shear scheme passed to
                   assemble_segment_indep (production ring choice ties only
                   gamma_23).
    lam_space      str, default 'elem'; drilling-constraint multiplier space
                   passed to assemble_constraint.
    return_fields  bool, default False; if True also return V0, V1 warping
                   fields (multiplier rows stripped).

    If return_fields=True: 3-tuple (C6r, V0[:M], V1[:M]) where V0 (M,4) and V1
    (M,4) are the zeroth- and first-order warping fields (M=6*Nd dof rows,
    Lagrange-multiplier rows stripped), including the drilling om3 boundary
    values, for the Dirichlet transfer.

    Builds a one-quad-deep prismatic strip: nodes = [rx; rx + h*e_ax], dof_map
    wraps the top row onto the bottom (concatenate(arange(m),arange(m))),

    (per-unit-length). Embeds the validated 5-DOF rigid kernel/constraint
    (build_C_Psi from opensg_jax.fe_jax.msg_rm_timo on rx[:,cross]) into 6

    (Psi6[3::6,3]*=-1). Forms the augmented KKT [Dhh_a, Gc.T; Gc, 0] plus

    finalize_v1_and_compute_deff (from opensg_jax.fe_jax.msg_solver, JAX).
    Deff = Dee + V0.T@Dhe_a. C6r symmetrized 0.5(C+C.T). Calls
    assemble_segment_indep, assemble_constraint, NDOF6.

 compute_curvatures   [public]
................................................................................
  compute_curvatures(centroids, e1s, e2s, e3s, elems):

  Compute the full per-element initial-curvature tensor (Yu thesis eq 5.10) by
  finite-differencing the flat-facet local frame across neighbouring elements,
  returning zero on flat facets.
  Args:
    centroids      (Ne,3) array-like of element centroid coords; np.asarray'd
                   internally.
    e1s            (Ne,3) per-element axial unit vector (beam-axis tangent).
    e2s            (Ne,3) per-element hoop unit vector (in-plane tangent).
    e3s            (Ne,3) per-element normal unit vector.
    elems          (Ne, nnpe) integer connectivity; used to build a
                   node->element adjacency for neighbour finding.
  Returns:
    dict with keys 'k11','k12','k21','k22','k13','k23', each a (Ne,) float
    array. Axial neighbours (|d.e1|>0.5|d|) give k11=de3.e1/dx1,
    k12=de3.e2/dx1, k13=de1.e2/dx1; hoop neighbours (|d.e2|>0.5|d|) give
    k21=de3.e1/ds, k22=de3.e2/ds, k23=de1.e2/ds. Prismatic circle ->
    (0,0,0,-1/R,0,0).
  Notes:
    Builds node2el defaultdict(list). For each element collects neighbours
    sharing a node, classifies each as axial or hoop by centroid-displacement
    direction, and averages (np.mean) the per-neighbour finite-difference
    estimates. Skips displacements with norm < 1e-12.

 compute_k22   [public]
................................................................................
  compute_k22(centroids, e2s, e3s, elems, flat_tol=1e-3, k22_max=None, cop_tol=0.5):

  Estimate per-element hoop normal curvature k22 = d(e3)/ds . e2 (== -1/R for
  a circle with inward e3) from hoop-aligned neighbours, with guards to keep
  genuine smooth curvature but reject sharp folds/junctions.
  Args:
    centroids      (Ne,3) element centroid coords (line elements for a 1-D
                   contour or quads for a 2-D segment).
    e2s            (Ne,3) per-element hoop unit tangent.
    e3s            (Ne,3) per-element normal.
    elems          (Ne, nnpe) integer connectivity for node->element
                   adjacency.
    flat_tol       float, default 1e-3. |k22| below this snaps to exactly 0 (~
                   R > 1 km).
    k22_max        float or None, default None. If set, clips |k22| to this
                   cap (for sharp trailing-edge spikes).
    cop_tol        float, default 0.5. Coplanarity filter: a neighbour counts
                   only if e3s[ej].e3s[ei] >= cop_tol (~ within 60 deg); a
                   near-perpendicular fold is rejected as a discontinuity, not
                   curvature.
  Returns:
    k22: (Ne,) float array of per-element hoop curvature.
  Notes:
    Per element: keeps only coplanar hoop-direction neighbours
    (|disp.e2|>=0.5|disp|), forms per-neighbour estimate (e3s[ej]-e3s[ei]).e2
    / ds, combines with np.median (robust to stray branch), then flat-snap and
    optional clip. Rejecting folds prevents spurious curvature spikes that
    break section symmetry (square GA2!=GA3) and blow up boundary-ring
    torsion.

  _quad_ops(X, e1, e2, e3, xi, eta, k22, cross=(1, 2), full_curvature=False):

  Build the prismatic-reduction Reissner-Mindlin membrane/bending strain
  operator BDq, transverse-shear operator BGq (pre-tying), and shear-warping
  operator BLq, plus geometry, at a quad Gauss point.
  Args:
    X              (4,3) node coordinates of the quad.

    e2             (3,) hoop unit frame vector (d/ds direction).
    e3             (3,) normal unit frame vector (accepted for API symmetry;

                   coordinate indices (beam axis is the remaining index):
                   (1,2) for axis=x cylinder, (0,1) for axis=z BAR-URC.

                   NO-OP in RM (no second derivative to add).
  Returns:

    [eps11,eps22,2eps12,k11,k22,2k12]; BGq (2,20) transverse-shear rows

  Assemble the GENERAL 3D-taper RM strain operators (membrane, curvature with
  Lambda drilling, and full drilling-eliminated transverse shear) plus the
  macro map at a quad Gauss point.
  Args:
    X              (4,3) quad node coordinates.

                   indices.
    ax             int or None, default None. Beam-axis index; if None
                   inferred as the index not in cross.
    kg             float, default 0.0. Hoop geodesic curvature (taper); scaled
                   by module KG_ABL before use.
  Returns:
    (BDe, BDh, BDl, BGe, BGh, BGl, dA): BDe (6,4) macro map Gamma_eps, BDh
    (6,20) fluctuation Gamma_h, BDl (6,20) x1-derivative Gamma_l; BGe (2,4),

 _surf_frame   [private/internal]

  _surf_frame(X, e3_mat, xi, eta, cross, ax):

  Build the geometric surface frame (a1 axial-tilted, a2 hoop, n normal) and
  the in-plane derivative operator at (xi,eta) for the GENERAL taper RM
  operator, returning the direction-cosine set.
  Args:
    X              (4,3) quad node coordinates.
    e3_mat         (3,) stored material normal, used to sign-match n (keeps
                   mesh's inward/outward choice).
    xi             float hoop parametric coord.
    eta            float axial parametric coord.
    cross          tuple(int,int) cross-section coordinate indices (global).
    ax             int global beam-axis index.
  Returns:
    (N, D1, D2, dA, c): N (4,) shape values; D1 (4,) d/dzeta1 (axial), D2 (4,)
    d/dzeta2 (hoop) per-node operators; dA float surface Jacobian; c dict of
    direction cosines {x11,x21,x31,x12,x22,x32,y1,y2,y3,x2,x3}.
  Notes:
    a2=Jxi/|Jxi|; a1=Gram-Schmidt of Jeta against a2 (tilted by taper ->
    nonzero x21,x31); n=cross(a1,a2), flipped (with a1) if n.e3_mat<0 to keep
    right-handed (a1,a2,n). Derivatives: d=(inv(G.T)@[dNx;dNe]) with
    G=[[a1.Jxi,a1.Jeta],[a2.Jxi,a2.Jeta]]; rows D1,D2. Cosines
    x_{i;alpha}=global-i component of a_alpha, y_i=normal components, x2/x3 =
    cross-section coords of the point.

  _omega3_ops(N, D1, D2, c, k22):

  Build the drilling (omega_3 and omega'_3) operator row-vectors of paper
  Appendix A.3/A.5 over the element's 20 DOFs, for the drilling-eliminated
  general RM formulation.
  Args:
    N              (4,) bilinear shape values.
    D1             (4,) d/dzeta1 per-node operator.
    D2             (4,) d/dzeta2 per-node operator.

  Returns:
    (OM3h, OM3l, OM3e, OM3pl): OM3h (20,) acting on w (shape part + -(C3b/C33)
    omega_b), OM3l (20,) acting on w' (=x11 x_{i;2}-x12 x_{i;1} weighted),
    OM3e (4,) acting on beam strain eb (only k1 entry via swept-area Rn1/Rn2),
    OM3pl (20,) = omega'_3 operator on w'. All scaled by OM3_SIGN (+1).
  Notes:
    C33=_c33_floor(y3). Rn1=x2 x31 - x3 x21, Rn2=x2 x32 - x3 x22 (A.4). S/(2
    C33) structure: OM3e[1]=(x11 Rn2 - x12 Rn1)/(2C33); per-node
    OM3h[i]+=(x_{i;2}D1-x_{i;1}D2)/(2C33), OM3l[i]+=(x11 x_{i;2}-x12
    x_{i;1})N/(2C33); OM3pl mirrors the shape part; algebraic -(C3b/C33)
    omega_b into DOFs 3,4 of OM3h and OM3pl (om'_b).

 _lambda_ops   [private/internal]
................................................................................
  _lambda_ops(N, D1, D2, c, k22, kg, alpha):

  Build the Lambda_alpha = omega'_3 x_{1;alpha} + omega_{3|alpha} operators
  (paper A.6) that enter ONLY the curvature-strain rows, with geometric
  coefficient derivatives modeled by the Darboux frame of the hoop line.
  Args:
    N              (4,) shape values.
    D1             (4,) d/dzeta1 operator.
    D2             (4,) d/dzeta2 operator.
    c              dict of direction cosines from _surf_frame.
    k22            float hoop normal curvature (d e3/ds . e2).
    kg             float hoop geodesic curvature (d e1/ds . e2; 0
                   prismatic/flat).
    alpha          int in {1,2}: which surface direction the Lambda is
                   differentiated along (selects x_{1;alpha}, Da=D1 or D2, and
                   the coefficient-derivative model).
  Returns:
    (Lh, Ll, Le): Lh (20,) acting on w, Ll (20,) acting on w', Le (4,) acting
    on beam strain eb.
  Notes:
    Darboux model: alpha==1 (axial) frame derivatives=0 but coordinate derivs
    survive (x_i)_{;1}=x_{i;1}; alpha==2 (hoop) (a1)_{;2}=kg a2,
    (a2)_{;2}=-k22 n - kg a1, (n)_{;2}=k22 a2. Builds C33_a, Rn*_a,
    dinv=d/dz[1/(2C33)]. Adds: t1 kappa1 coeff group into Le[1]; t4
    x_{1;alpha} omega'_3 via Ll += x1a*OM3pl; t3 w'_{i|alpha}; t5 w' coeff
    group; t7 w_{i|alpha} coeff group; t8 shape/coeff -(C3b/C33)
    omega_{b|alpha}. Drops 2nd SG derivatives w_{i|1|alpha}, w_{i|2|alpha} and
    kappa'_11 (documented). Calls _omega3_ops. Every term carries OM3_SIGN.

  _strip(rx3, cells, ax):

    m+b, m+a] (m = number of ring nodes). Converts rx3 to float np array
    first.

 _rm_shell_strain   [private/internal]
................................................................................
  _rm_shell_strain(B, e, xi, st_m, aA, aB, s2_scheme="mitc4_g23"):

  Evaluate the 8 RM shell strains at element e and arc position xi: the 6
  membrane/bending strains s6=[e11,e22,2e12,k11,k22,2k12] and the 2 transverse
  shears s2=[2g13,2g23].
  Args:
    B              bundle dict from build_rm_bundle (uses B['strip'],
                   B['re3'], B['k22'], B['cross'], B['ax'], B['red_cells']).
    e              int element index into the ring.
    xi             float arc coordinate in [0,1] along the element; mapped to
                   natural coord xq = 2*xi - 1 in [-1,1].
    st_m           ndarray (6,) macro membrane/bending strain vector (from
                   _macro_recovery).
    aA             ndarray (6m,) warping field w = a1 + a2 (from
                   _macro_fields); gathered to element DOFs as wA.
    aB             ndarray (6m,) warping-derivative field w' = a4 + a3 (from
                   _macro_fields); gathered as wB.
    s2_scheme      str tying scheme for the RECOVERED transverse shear s2,
                   default 'mitc4_g23' (matches homogenization: only g23 tied,
                   g13 raw carries flat-wall drilling). 'mitc4_both' returns
                   drilling-free tied shear on both rows (physical wall
                   shear).
  Returns:
    tuple (s6, s2): s6 = ndarray (6,) = [e11,e22,2e12,k11,k22,2k12]; s2 =
    ndarray (2,) = [2g13,2g23].
  Notes:
    Gathers element geometry Xe=nodes[quads[e]] and e3e=B['re3'][e]. Calls
    quad_ops_indep(Xe,e3e,xq,0.0,k22[e],cross,ax) returning operators
    BDe,BDh,BDl,BGe,BGh,BGl,DRe,DRh,DRl,dA, and
    _mitc_shear_indep(...,scheme=s2_scheme) for the tied shear operator BGt.
    Builds a 24-length DOF gather index g = [c0*6:c0*6+6, c1*6:+6, c1*6:+6,
    c0*6:+6] (the a1,a2,a4,a3 ordering) where c0,c1 = red_cells[e]. Computes
    s6 = BDe@st_m + BDh@wA + BDl@wB and s2 = BGe@st_m + BGt@wA + BGl@wB (BGt
    replaces BGh for the MITC-tied transverse shear).

  Compute the macro beam strain and the two RM warping combination fields (aA,

  recover_shell_strains(bundle, beam_force_vabs=None, beam_strain=None, xi_eval=None)

  Step 1 of dehomogenization: recover the 6 plate/shell strains along the
  cross-section from a beam force or prescribed beam strain, using the MSG
  shell strain operators plus the EB (V0) and shear (V1) warping fields.
  Args:
    bundle         dict from msg_hermite.solve_tw_from_yaml; reads Timo (6,6),
                   V0 (N,6), V1 (N,6), red_cells (E,2), corners (Nc,2), k22
                   (E,), L (E,), xd2 (E,), xd3 (E,), xi_q (Pq,).
    beam_force_vabs (6,) applied beam force in VABS order [F1,F2,F3,M1,M2,M3];
                    provide exactly one of this or beam_strain. Strain
                    recovered as inv(Timo)@force.
    beam_strain    (6,) prescribed beam strain in VABS order; taken directly
                   as st when given.
    xi_eval        (P,) optional arc positions in [0,1] to evaluate; default =
                   bundle['xi_q'] Gauss points. Use [0.0,1.0] for the two end-
                   node values.
  Returns:
    dict: 'shell_strain' (E,P,6)
    [eps11,eps22,gamma12,kappa11,kappa22,kappa12]; 'shell_strain_elem' (E,6)
    per-element mean over P; 'x2','x3' (E,P) arc-point coords; 'xi' (P,)
    positions; 'tang' (E,2) unit arc tangent [xd2,xd3]; 'macro' (6,) beam
    strain st; 'st_m' (4,) EB macro strain.
  Notes:
    Raises ValueError unless exactly one of beam_force_vabs/beam_strain is
    given (XOR check via (beam_strain is None)==(beam_force_vabs is None)).
    Calls _macro_recovery. Builds four warping combos: a1=V0@st_m (EB warping
    w_1), a2=V1@st_cl1 (shear warping w1s_1), a3=V1@st_cl2 (shear warping
    w1s_2), a4=V0@st_cl1 (EB-of-derivative w_2). Per element extracts 12-dof
    gather g = [c0*6:c0*6+6, c1*6:c1*6+6], calls
    hermite_strain_operators(n0,n1,k22,L,xd2,xd3,xi) -> (eps_h,eps_l,eps_e,_)
    and assembles s = eps_h(a1[g]) + eps_e(st_m) + eps_h(a2[g]) + eps_l(a4[g])
    + eps_l(a3[g]). x2/x3 are linear interpolation (1-xi)*n0 + xi*n1. Mixes
    JAX (jnp) operator calls with numpy accumulation.

 _recov   [private/internal]
................................................................................
  _recov(st)

  Build the 6x6 Timoshenko derivative-recovery matrix from a beam-strain
  vector, matching the OpenSG solid `utils.recov`. Encodes the cross products
  that couple force/moment components under axial differentiation.
  Args:
    st             (6,) array-like beam strain [eps11, gamma12, gamma13, k1,
                   k2, k3]; only components st[0..5] are read to fill the skew
                   blocks.
  Returns:
    R : (6,6) numpy array. Upper-left 3x3 skew block from st[3],st[4],st[5];
    R[3:6,3:6] copies R[0:3,0:3]; lower rows add st[0],st[1],st[2] cross
    terms.
  Notes:
    Pure numpy, no JAX. R[0,1]=st[5],R[0,2]=-st[4];
    R[1,0]=-st[5],R[1,2]=st[3]; R[2,0]=st[4],R[2,1]=-st[3]; then
    R[3:6,3:6]=R[0:3,0:3]; R[3,1]=st[2],R[3,2]=-st[1];
    R[4,0]=-st[2],R[4,2]=st[0]; R[5,0]=st[1],R[5,1]=-st[0]. Called by
    _macro_recovery with the fixed unit strain [1,0,0,0,0,0].

 _macro_recovery   [private/internal]
................................................................................
  _macro_recovery(Ceff_srt, st)

  From the Timoshenko 6x6 and a prescribed beam strain, produce the EB macro
  strain st_m and the first/second Timoshenko derivative-recovery terms
  st_cl1/st_cl2, each Q-corrected for the EB<->shear coupling. Mirrors the
  corrected OpenSG-FEniCS stress_recov.local_strain.
  Args:
    Ceff_srt       (6,6) Timoshenko stiffness matrix (VABS/sorted order).
    st             (6,) prescribed beam strain [eps11, gamma12, gamma13, k1,
                   k2, k3]; cast to float via np.asarray.
  Returns:
    (st, st_m, st_cl1, st_cl2): st the unchanged (6,) input; st_m (4,)
    Q-corrected EB macro strain [eps11,k1,k2,k3]; st_cl1 (4,) first-derivative
    recovery; st_cl2 (4,) second-derivative recovery.
  Notes:
    Comp=inv(Ceff_srt). st_m initialized as st[[0,3,4,5]]. F_1d=Ceff_srt@st is
    the equivalent beam force. R1=_recov([1,0,0,0,0,0]) is the FIXED recovery
    operator (not recov(st)). Iterates F1=R1@F_1d, F2=R1@F1, F3=R1@F2;
    stTk=Comp@Fk; st_clk picks indices [0,3,4,5]; gammak picks [1,2]. Applies
    _Q@gamma corrections: st_m+=_Q@gamma1, st_cl1+=_Q@gamma2,
    st_cl2+=_Q@gamma3, where _Q is the module-level 4x2
    [[0,0],[0,0],[0,-1],[1,0]]. The old recov(st+e0)/R2/R3 terms are
    deliberately dropped.

 finalize_v1_and_compute_deff   [public]

  @jax.jit
def finalize_v1_and_compute_deff(V1s_raw, V0, D_eff, V0DllV0, DhlV0, DhlTV0Dle, Psi, Dc)

  Project the raw V1 shear-warping field onto the constraint null space and
  assemble the final sorted 6x6 Timoshenko stiffness via the Schur/static-
  condensation of the shear DOFs.
  Args:
    V1s_raw        (N, n_cases) unprojected V1 shear-warping solution from the
                   KKT solve.
    V0             (N, n_cases) EB fluctuation field.
    D_eff          (4,4) effective EB stiffness (Hessian of the EB energy);
                   symmetrized internally.
    V0DllV0        (n_cases,n_cases) from prepare_v1_rhs.
    DhlV0          (N,n_cases) from prepare_v1_rhs.
    DhlTV0Dle      (N,n_cases) = Dhl.T@V0+Dle_dense, from prepare_v1_rhs.
    Psi            (N, n_con) null-space basis.
    Dc             (N, n_con) constraint matrix.
  Returns:
    (S, B_tim, C_tim, V1s): S (6,6) sorted Timoshenko stiffness
    [EA,GA12,GA13,GJ,EI2,EI3]; B_tim (4,4)=DhlTV0Dle.T@V0; C_tim (4,4)
    symmetrized shear-energy matrix; V1s (N,n_cases) projected shear-warping
    field.
  Notes:
    JAX-jitted. V1s=V1s_raw - Psi@inv(Dc.T@Psi)@(Dc.T@V1s_raw). D_eff
    symmetrized 0.5*(D_eff+D_eff.T) to remove the antisymmetric taper artifact
    (Dirichlet segment) that inflates C26/C35 shear-bending coupling.
    Ainv=inv(D_eff). C_unsym=V0DllV0 + V1s.T@(DhlV0+DhlTV0Dle);
    C_tim=0.5*(C_unsym+C_unsym.T). Q_base=[[0,0],[0,0],[0,-1],[1,0]] (same as
    _Q); Q_tim=Ainv@Q_base. Ginv=Q_tim.T@(C_tim - B_tim.T@Ainv@B_tim)@Q_tim;
    G_tim=inv(Ginv) (shear stiffness GA12/GA13). Y_tim=B_tim.T@Q_tim@G_tim;
    A_tim=D_eff+Y_tim@Ginv@Y_tim.T. S assembled placing A_tim into rows/cols
    {0,3,4,5}, G_tim into {1,2}, Y_tim into the shear-extension/shear-bending
    couplings; row 0=[A00,Y[0,:],A[0,1:4]], rows 1:3=[Y.T[:,0],G,Y.T[:,1:4]],
    rows 3:6=[A[1:4,0],Y[1:4,:],A[1:4,1:4]].

  compute_ABD_matrix(thick, angles_deg, mat_names, material_db, n_per_layer=1, return_warping=False, elem_order=2, shear_refined=False, z_ref=0.0):

  Compute the plate ABD stiffness of a laminate via the MSG 1D through-
  thickness SG (variational, quadratic-Lagrange, exact for uniform layers),
  optionally the 8x8 Reissner-Mindlin stiffness with the transverse-shear
  block, plus mass and warping fields.
  Args:
    thick          list[float] — layer thicknesses, ordered bottom(OML) to
                   top(IML)
    angles_deg     list[float] — fiber angle (deg) per layer
    mat_names      list[str] — material-db key per layer
    material_db    dict {name: {E:[3], G:[3], nu:[3], rho or density: float}}
                   — material properties
    n_per_layer    int, default 1 — number of Lagrange sub-elements per
                   physical layer (1 is exact for uniform layers)
    return_warping bool, default False — if True also return the warping dict
                   for plate dehomogenization
    elem_order     int, default 2 — element polynomial order p (2=quadratic
                   3-node, 3=cubic); p+1 nodes, max(3,p+1)-pt Gauss
    shear_refined  bool, default False — if True return the 8x8 RM plate
                   stiffness [[A,B,0],[B,D,0],[0,0,G]] instead of the 6x6
    z_ref          float, default 0.0 — through-thickness coordinate of the
                   reference surface measured from the bottom/OML face; the 1D
                   mesh origin is shifted by z_ref (0=OML, sum(thick)/2=mid,
                   sum(thick)=IML)
  Returns:
    Without return_warping: (stiff, mass). stiff = (6,6) MSG ABD (default) or
    (8,8) RM stiffness when shear_refined. mass = [mu, mu*xm3, i22_m]
    (mass/area, first moment mu*centroid, second moment). With return_warping:
    (stiff, mass, warp) where warp is a dict {V0:(ndofs,6), node_x:(n_node,),
    elem_layer:(n_elem,), C_layers:list of (6,6), elem_order:p, angles:list}.
  Notes:
    Variational MSG: gamma_h=B, macro map Ge (rows: eps11<-1+x*k11 col0/3,
    eps22<-1+x*k22 col1/4, gamma12<-1+x*k12 col2/5). Assembles K=B^T C B,
    F_load=B^T C Ge, D_ee=Ge^T C Ge over 3-node quad elements with Gauss.
    Rigid-body nullspace (3 constant translations) removed by PROJECTION
    P=I-QQ^T (not penalty, to avoid swamping near-zero soft plies like gelcoat
    E=10) with K_proj=P K P + beta QQ^T, beta=max|diag K|.
    V0=solve(K_proj,-(P@F_load)) then P@V0; D_eff=D_ee+V0^T F_load. Calls
    rotated_stiffness_6x6, _plate_B; shear_refined calls
    transverse_shear_stiffness and plate_8x8. rho read via 'rho' then
    'density' fallback.

  transverse_shear_stiffness(thick, angles, mat_names, mat_db, nz_per_ply=40, coupled=True):

  Compute the laminate 2x2 transverse-shear stiffness (the G block of the RM
  8x8 plate law) by the FSDT/3D-equilibrium shear-flow method with
  complementary-energy equivalence, plus a shear-stress recovery function.
  Args:
    thick          list[float] — layer thicknesses
    angles         list[float] — fiber angles (deg) per layer
    mat_names      list[str] — material-db key per layer
    mat_db         dict — {name:{E,G,nu}} material properties
    nz_per_ply     int, default 40 — through-thickness sample points per ply
                   (endpoint excluded)
    coupled        bool, default True — True uses the coupling-aware MSG form
                   (full 2x2 from Gbar^{-1}(z)); False uses the per-direction
                   diagonal FSDT/Whitney(1973) energy-equivalence stiffness
  Returns:
    (Gmat, recover, aux): Gmat (2,2) transverse-shear stiffness
    (rows/cols=[2g13,2g23]); recover(z,V)->[sigma13,sigma23] callable for
    stress recovery; aux = (zs, gx/Dx, gy/Dy) with zs (N,) sample coords,
    gx/Dx (N,) and gy/Dy (N,) the normalized shear-flow shapes ghat.
  Notes:
    Per direction (nested function `direction`): neutral axis z_na = sum(Q z
    dz)/sum(Q dz); D_eff = sum(Q zc^2 dz); shear flow g = cumsum(Q zc dz);
    Whitney F = D_eff^2 / sum(g^2/G dz). coupled=True builds M_ab = sum(ghat_a
    ghat_b [Gbar^{-1}]_ab dz) with ghat=g/D_eff and Gmat = inv(M);
    coupled=False gives diag(Fx,Fy). Coincides when Gbar diagonal (G13=G23).
    Validates isotropic -> (5/6) G h. Calls _ply_Q_and_G. dz = htot/len(zs)
    uniform.

 hermite_strain_operators   [public]

  hermite_strain_operators(n0, n1, k22, L, xd2, xd3, xi_q):

  Build the per-element MSG shell strain operators (eps_h bending fluctuation,
  eps_l Timoshenko shear-warping, eps_e macro EB map) and the macro-strain map
  Ge at the quadrature points, for a Hermite C1 element. Single source of
  truth shared by assembly and dehomogenization.
  Args:
    n0             (2,) array — first corner coordinate (y2,y3)
    n1             (2,) array — second corner coordinate
    k22            float — element curvature (-1/R; flat=0)
    L              float — element arc length
    xd2            float — tangent component dx2/ds
    xd3            float — tangent component dx3/ds
    xi_q           (Q,) array — reference quadrature coordinates in [0,1]
  Returns:
    (eps_h, eps_l, eps_e, Ge): eps_h(u) callable u(12,)->(Q,6); eps_l(u)
    callable (Q,6); eps_e(ue) callable ue(4,)->(Q,6) = Ge@ue; Ge (Q,6,4) macro
    EB strain map. Shell strain order
    [eps11,eps22,gamma12,kappa11,kappa22,kappa12].
  Notes:
    Uses hermite_shape_functions; DOF component indices _I1/_I2/_I3 pick
    w1/w2/w3 value+slope from the 12-DOF vector. Rn = x2 xd3 - x3 xd2 (moment
    arm). Ge nonzeros: eps11<-EA(col0)+kappa about x3 (col2)/x2(col3),
    gamma12<-Rn*twist(col1), kappa11<-xd2/xd3(col2/3), kappa12<--2+k22/2*Rn
    (col1). eps_h: eps22=xd2 dw2+xd3 dw3, gamma12=dw1, kappa22=-k22 xd2 dw2 +
    xd3 d2w2 - k22 xd3 dw3 - xd2 d2w3, kappa12=(k22/2) dw1. eps_l: eps11=w1,
    eps22=xd2 dw2+xd3 dw3, gamma12=xd2 w2+xd3 w3, kappa12=2 xd3 dw2 - 2 xd2
    dw3 + (k22/2)(xd2 w2+xd3 w3). Pure JAX (differentiable).

  Assemble the global MSG-shell system matrices (Dhh, Dhe, Dee, Dll, Dhl, Dle)

  elements and scattered into sparse COO.
